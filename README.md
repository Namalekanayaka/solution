# MindCheck - Mental Health Support Web App (Frontend)

## Project Overview
**MindCheck** is an advanced, responsive, and interactive **mental health support web application** built with **React** and **Tailwind CSS**.  
It allows users to:  
- Take a **mental health quiz**  
- View **personalized results** with color-coded feedback  
- Track **past quiz results and mood history**  
- Access **resources and emergency hotlines**  
- Optionally **register/login** to save results in future (backend-ready)  

> ⚠️ Disclaimer: This project is for educational purposes only. It is **not a medical diagnosis tool**. Users should consult professionals for medical advice.

## Tech Stack
- **React 18** (Vite) – Frontend framework  
- **Tailwind CSS** – Utility-first styling for responsive design  
- **React Router Dom** – Single Page Application routing  
- **Axios** – API requests to backend (or local JSON)  
- **Framer Motion** – Animations for smooth UX  
- **Recharts** – Charts for mood tracking  
- **Lottie-React** – Animated SVGs for engagement  
- **localStorage** – Temporary storage for quiz results
